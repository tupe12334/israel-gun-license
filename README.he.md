# רישיון לנשיאת אקדח — הגשת בקשה בעזרת AI

סביבת עבודה של Claude Code שמנחה אותך בתהליך הגשת בקשה לרישיון לנשיאת אקדח דרך פורטל gov.il.

## איך זה עובד

1. שכפל את הריפו ופתח בClaude Code
2. אמור לקלוד: "עזור לי להגיש בקשה לרישיון נשק"
3. קלוד יבדוק את זכאותך לפי חוק כלי ירייה
4. קלוד יאסוף את הפרטים הנדרשים
5. שרת Playwright MCP שולט על חלון דפדפן אמיתי
6. קלוד ינווט בפורטל gov.il, ימלא את טופס הבקשה ויגיש אותה
7. קלוד ינחה אותך בשלבים הבאים — בדיקה רפואית ומעקב סטטוס

אין צורך בידע תכנותי.

## דרישות מוקדמות

| דרישה | פרטים |
|-------|-------|
| [Claude Code](https://claude.ai/code) | אפליקציה, תוסף VS Code, או CLI |
| [Playwright MCP](https://github.com/microsoft/playwright-mcp) | `npm install -g @playwright/mcp` |
| Node.js 18+ | נדרש עבור Playwright MCP |
| תעודת זהות ישראלית | נדרש לאימות בgov.il |

## שימוש

```bash
git clone https://github.com/tupe12334/israel-gun-license.git
cd israel-gun-license
claude .
```

לאחר מכן אמור לקלוד:

> "עזור לי להגיש בקשה לרישיון נשק"

## מה קלוד לא יכול לעשות

- **בדיקה רפואית** — חייבת להתבצע פנים אל פנים אצל רופא מוסמך
- **בדיקת רקע** — מבוצעת אוטומטית על ידי המשטרה
- **הדרכת נשק** — נדרשת טווח ירי מאושר (אם חל)
- **ההחלטה הסופית** — בסמכות ענף רישוי נשק

## אזהרה

מתן מידע כוזב בבקשה לרישיון נשק הוא עבירה פלילית לפי החוק הישראלי. אתה אחראי על נכונות המידע שאתה מספק.
